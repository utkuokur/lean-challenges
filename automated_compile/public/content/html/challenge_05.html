Challenge 5: Erdős–Hajnal Conjecture

Definition 1 (Cliques and Independent sets).

Given a simple graph G, a set of vertices of G is a clique if its vertices are pairwise adjacent. It is an independent set if its vertices are pairwise non-adjacent.

Definition 2 (H-free graphs).

Given graphs G and H, we say that G is H-free if it does not contain an induced subgraph isomorphic to H.

Conjecture 1.1 (Erdős–Hajnal Conjecture [1]).

For each graph H, there is a constant cH>0 such that every H-free graph G with n vertices has a clique or independent set of size at least ncH.

Given r∈ℕ, we denote the path with r vertices by Pr. We are interested in the following parametrized version of the conjecture, where H=Pr is a path.

Conjecture 1.2 (Erdős–Hajnal Conjecture for paths).

Let r∈ℕ. There is some cr>0 such that every Pr-free graph G with n vertices has a clique or independent set of size ncr.

The conjecture has been proved for r=5 by Nguyen, Tung, Scott and Seymour in [2]
