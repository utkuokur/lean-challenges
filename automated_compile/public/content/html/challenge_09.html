Challenge 9: Union-Closed Sets Conjecture

Definition 1 (Union-closed set).

A set ℱ of sets is union-closed if for any two sets in ℱ, their union is also in ℱ. For a nonempty finite set ℱ, the density of an element x with respect to ℱ is defined to be

|{F∈ℱ:x∈F}||ℱ|
Conjecture 1.1 (Union-closed sets conjecture [1]).

For every finite union-closed set ℱ∉{∅,{∅}}, there exists an element x of density at least one half.

We are interested in the following parametrized version of the conjecture.

Conjecture 1.2 (Scaled union-closed sets conjecture).

Let r∈ℕ. For every finite union-closed set ℱ∉{∅,{∅}}, there exists an element x of density at least 12−1r+2.

For r=0 the bound equals 0 and the statement is elementary, while as r→∞ the bound approaches 12 and the statement recovers the full conjecture in the limit. The best published lower bound on the density (≈0.382, obtained via the entropy method) settles the cases r≤6; the conjecture is open for all r≥7.
