Challenge 8: Ryser’s Hypergraph Conjecture

Definition 1 (r-partite).

Given r∈ℕ, an r-uniform hypergraph ℋ is r-partite if there is a partition

V⁢(ℋ)=⋃i=1rPi

of the vertex set of ℋ such that |Pi∩e|=1, for each e∈E⁢(ℋ) and i∈[r].

Definition 2 (Cover number).

Given a hypergraph ℋ, a set C of vertices of ℋ is a vertex cover if it contains a vertex of every edge. The smallest cardinality of a vertex cover of ℋ is called the cover number of ℋ. It is denoted by τ⁢(ℋ).

Definition 3 (Matching number).

The matching number of a hypergraph ℋ is the largest cardinality of a set of pairwise disjoint edges of ℋ. It is denoted by ν⁢(ℋ).

Conjecture 1.1 (Ryser’s Conjecture [2]).

Let r∈ℕ with r≥2. Every r-partite r-uniform hypergraph ℋ satisfies:

τ⁢(ℋ)≤(r−1)⋅ν⁢(ℋ)

For r=2, it is a classical theorem of Kőnig, and for r=3 it was proved by Aharoni [1] using the topological method. The hypothesis r≥2 is necessary: for r=1 the bound reads τ⁢(ℋ)≤0, which already fails for the single-vertex single-edge hypergraph (where τ=ν=1).
